# Security Policy

## Supported Versions
The latest version is in development, there will be a large number of security errors or fatal errors, and the document will be modified and released before the official release.

| Version | Supported |
| ------- | --------- |
| latest  | :x: |

## Reporting a Vulnerability
If you discover a security vulnerability, please do not disclose it publicly.
Instead, open a GitHub Private Vulnerability Report (if available) or contact the maintainer privately.
I will acknowledge valid reports as soon as possible and work on a fix before public disclosure.
Thank you for helping keep this project secure.
